# 快速上手

#### 引入组件库，拿到一些组件实例
```js
import {Button,ButtonGroup,Collapse,CollapseItem} from 'sim-ui-use';
```






#### 特别注意：:tada: :100:
::: warning
一些必要的资源：引入组件库CSS
:::

```js
import 'sim-ui-use/dist/index.css'
```






#### 注册组件
```js
export default {
      components: {
        Button,
        ButtonGroup,
        Collapse,
        CollapseItem,
      }
}
```
